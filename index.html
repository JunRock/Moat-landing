02

또 "저 회비 냈는데요?"

같은 질문에 같은 답을 매달 다섯 번씩. 운영진의 시간이 그만큼 사라집니다.

03

공지가 묻혀버려요

정모 시간, 투표 안내, 회비 마감이 단톡방 메시지 사이에 묻혀버립니다.

운영, 이렇게 달라져요

흩어진 도구를
하나의 앱으로.

정모부터 자동 알림까지. 동호회 운영에 필요한 전부를 홈에서 바로 확인하세요.

정모 일정

다음 정모,
모두가 알아요.

오늘 기준 가장 가까운 다음 정모가 홈 화면 가장 위에 자동으로 떠요. 정모 전날 아침에는 참석자에게 자동 알림이 가니까, 다시 단톡방에 공지할 필요가 없어요.

  • 가장 가까운 정모, 홈에 자동 표시
  • 정모 전날 오전 10시, 참석자에게 자동 알림
  • 참석·취소도 탭 한 번으로
회비 관리

한 달 정산,
자동으로 끝나요.

회비·게스트비·지출이 한 화면에. 납부일이 지나면 미납자에게 매일 자동 알림이 가요. 멤버는 토스로 한 번에 송금하면, 자동으로 기록돼요.

  • 회비·게스트비·지출 한 화면 관리
  • 미납자에게 매일 오후 9시 자동 알림
  • 토스로 1초 송금, 자동 기록
회원 관리

회원 정보가,
한눈에 들어와요.

조별·성별로 색상 구분된 회원 목록. 이름 검색이나 조·성별 필터로 한 번에 찾을 수 있어요. 모임장 위임이나 총무 지정도 같은 화면에서 끝나요.

  • 조·성별 통계 한 화면에
  • 이름 검색·조별 필터 빠른 조회
  • 모임장·총무 권한 부여까지 한 곳에서
코트 배치

오늘 누가
어느 코트에서?

정모 당일 코트별 게임 현황을 실시간으로 공유해요. 화이트보드와 메모지가 필요 없습니다. 운영진과 멤버 모두 모바일로 같이 봐요.

  • 코트별 게임 현황 실시간 공유
  • 화이트보드·메모지 불필요
  • 운영진·멤버 함께 확인
커뮤니티

공지와 사진이
한 곳에 모여요.

단톡방을 거꾸로 스크롤할 일이 없어요. 공지는 상단에 고정되고, 정모 사진은 한 앨범에 모여요. 새 글이 올라오면 자동 푸시 알림이 가니 멤버도 놓칠 일이 없고요.

  • 중요 공지는 상단 고정·자동 푸시 알림
  • 정모 사진 업로드
  • 게시판으로 자유로운 소통까지
투표

의견 모으기,
한 번에 끝.

투표가 단톡방 메시지에 섞이지 않고 따로 한 곳에 모여요. 정모 시간, 회비 인상, 단체복 디자인까지 — 안건마다 익명·복수 선택을 자유롭게 만들 수 있고요.

  • 복수 선택·익명 투표 옵션 지원
  • 마감일 설정 시 자동 종료
  • 새 투표 등록 시 멤버 전체에 푸시 알림
알림 발송

단톡방으로
안 보내도 돼요.

미납자, 정모, 새 공지, 투표까지. 필요한 알림은 앱이 자동으로 보내요. 단톡방 알림 피로도가 한결 줄어듭니다.

  • 미납자에게 매일 오후 9시 자동 알림
  • 정모 전날 오전 10시 자동 알림
  • 새 공지·투표도 자동 푸시 발송
